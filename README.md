# IntervalTimer  
![IntervalTimerApp](https://github.com/Lausiv1024/IntervalTimer/blob/master/doc/img/app.png)
複数個のタイマーを順番に動かすことができます<br>

## ダウンロード方法
1.右側のReleaseをクリック
2.最新バージョンをダウンロード
3.Zipを適当な場所に展開
あとは中のIntervalTimer.exeを起動するだけで使えます
